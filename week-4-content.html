Independant learning tasks

All of my planned colour schemes include the use of muted/pale/similar colours as opposed to bright, illuminous, overly contrasting colours as these can be very distracting. My use of a more simple colour pallet makes it easier to focus on the information on the page without having attention constantly drawn to loud use of colour. The background colour of the (body) text, however, will remain the same in every page, so as to create a seamless flow from page to page- there will be no jarring changes. All text prompts and text as a whole will also remain the same colours for each content page for the same reason as stated before.

  • Index:

    Index page will follow a coral/pale pink colour scheme. The headings gradually getting paler in colour- for example, main heading will be darkest, sub-headings will be lighter, etc. This creates a type of cascading effect that will allow the user to follow the flow of information naturally and as intended.
